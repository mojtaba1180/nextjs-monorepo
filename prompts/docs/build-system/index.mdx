# 🏗️ سیستم Build و Registry

این prompt برای کار با سیستم Build و Registry استفاده می‌شه.

## 📋 دستورالعمل

```
من می‌خوام با سیستم Build و Registry کار کنم. لطفا مراحل زیر رو به ترتیب انجام بده:

1. **تحلیل نیاز**: نیازهای Build و Registry رو مشخص کن
2. **بررسی ساختار**: ساختار فعلی رو بررسی کن
3. **اعمال تغییرات**: تغییرات مورد نیاز رو اعمال کن
4. **Build**: سیستم Build رو اجرا کن
5. **تست**: مطمئن شو همه چیز درست کار می‌کنه
```

## 📁 ساختار Build System

```
scripts/
└── build-registry.ts          # اسکریپت build registry

src/registry/
├── index.ts                   # تنظیمات registry
├── __index__.ts              # Auto-generated index
├── schema.ts                 # Schema definitions
└── custom-ui/                # فایل‌های کامپوننت‌ها
    └── [component].tsx

registry.json                 # فایل JSON registry
```

## 🔧 مراحل Build

### 1. تنظیمات Registry

```typescript
// src/registry/index.ts
export const registry = {
  items: [
    {
      name: "component-name",
      description: "توضیح کامپوننت",
      type: "registry:component",
      registryDependencies: undefined,
      files: [
        {
          path: "custom-ui/component-name.tsx",
          type: "registry:component",
          target: "components/component-name.tsx"
        }
      ],
      categories: ["components"],
      meta: {
        title: "ComponentName",
        description: "توضیح کامل کامپوننت"
      }
    }
  ]
}
```

### 2. اسکریپت Build

```typescript
// scripts/build-registry.ts
import { exec } from "child_process"
import { promises as fs } from "fs"
import path from "path"
import { rimraf } from "rimraf"

import { registry } from "@/registry/index"

async function buildRegistryIndex() {
  let index = `/* eslint-disable @typescript-eslint/ban-ts-comment */
/* eslint-disable @typescript-eslint/no-explicit-any */
// @ts-nocheck
// This file is autogenerated by scripts/build-registry.ts
// Do not edit this file directly.
import * as React from "react"

export const Index: Record<string, any> = {`
  
  for (const item of registry.items) {
    const componentPath = item.files?.[0]?.path
      ? `@/registry/custom-ui/${item.files[0].path}`
      : ""

    index += `
  "${item.name}": {
    name: "${item.name}",
    description: "${item.description ?? ""}",
    type: "${item.type}",
    registryDependencies: ${JSON.stringify(item.registryDependencies)},
    files: [${item.files?.map((file) => {
      const filePath = `registry/custom-ui/${typeof file === "string" ? file : file.path}`
      return typeof file === "string"
        ? `"${filePath}"`
        : `{
      path: "${filePath}",
      type: "${file.type}",
      target: "${file.target ?? ""}"
    }`
    })}],
    component: ${
      componentPath
        ? `React.lazy(async () => {
      const mod = await import("${componentPath}")
      const exportName = Object.keys(mod).find(key => typeof mod[key] === 'function' || typeof mod[key] === 'object') || "${item.name}"
      return { default: mod.default || mod[exportName] }
    })`
        : "null"
    },
    categories: ${JSON.stringify(item.categories)},
    meta: ${JSON.stringify(item.meta)},
  },`
  }

  index += `
  }`

  console.log(`#️⃣  ${registry.items.length} items found`)

  // Write registry index.
  rimraf.sync(path.join(process.cwd(), "src/registry/__index__.ts"))
  await fs.writeFile(path.join(process.cwd(), "src/registry/__index__.ts"), index)
}

async function buildRegistryJsonFile() {
  // 1. Fix the path for registry items.
  const fixedRegistry = {
    ...registry,
    items: registry.items.map((item) => {
      const files = item.files?.map((file) => {
        return {
          ...file,
          path: `registry/custom-ui/${file.path}`,
        }
      })

      return {
        ...item,
        files,
      }
    }),
  }

  // 2. Write the content of the registry to `registry.json`
  rimraf.sync(path.join(process.cwd(), `registry.json`))
  await fs.writeFile(
    path.join(process.cwd(), `registry.json`),
    JSON.stringify(fixedRegistry, null, 2)
  )
}

async function main() {
  try {
    console.log("🗂️ Building registry/__index__.ts...")
    await buildRegistryIndex()

    console.log("💅 Building registry.json...")
    await buildRegistryJsonFile()

    console.log("✅ Registry build completed!")
  } catch (error) {
    console.error(error)
    process.exit(1)
  }
}

main()
```

### 3. اجرای Build

```bash
# اجرای اسکریپت build
npx tsx scripts/build-registry.ts

# یا با pnpm
pnpm run build:registry
```

## 📝 قوانین Build

1. **Auto-Generation**: فایل `__index__.ts` همیشه auto-generated هست
2. **Schema Validation**: تمام items باید schema رو رعایت کنن
3. **Path Resolution**: path ها باید درست resolve بشن
4. **Lazy Loading**: کامپوننت‌ها باید lazy load بشن
5. **Error Handling**: خطاها باید handle بشن

## 🎯 انواع Build

### Build کامل
```bash
# Build تمام registry
npx tsx scripts/build-registry.ts
```

### Build جزئی
```bash
# Build فقط یک کامپوننت
npx tsx scripts/build-registry.ts --component=component-name
```

### Build با Clean
```bash
# Build با پاک کردن فایل‌های قبلی
npx tsx scripts/build-registry.ts --clean
```
