# DigiMoragheb Documentation

Welcome to the DigiMoragheb monorepo documentation. This documentation covers all the components, utilities, and applications in our monorepo.

## What's included

- **Custom UI**: A collection of custom UI components built specifically for the DigiMoragheb project
- **Applications**: Various applications including docs and expert applications
- **Packages**: Shared packages and utilities

To get started with the documentation, navigate to the section you're interested in using the sidebar.
