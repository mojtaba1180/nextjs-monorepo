# خوش آمدید به مستندات پروژه

این مستندات شامل راهنمای کامل استفاده از کتابخانه‌ها و کامپوننت‌های پروژه است.

## شروع سریع

برای شروع کار با کتابخانه‌ها، ابتدا آن‌ها را نصب کنید:

```bash
# نصب در پروژه‌های workspace
pnpm add @workspace/custom-ui

# یا در پروژه‌های خارجی
npm install @workspace/custom-ui
```

## ویژگی‌های کلیدی

- **کامپوننت‌های فرم**: کامپوننت‌های آماده برای React Hook Form
- **مدیریت خطا**: سیستم جامع نمایش و مدیریت خطاها
- **مدال‌ها**: کامپوننت‌های مدال responsive و قابل تنظیم
- **هوک‌های مفید**: مجموعه‌ای از هوک‌های کاربردی
- **Context ها**: مدیریت state سراسری برای خطا، مدال و toast
