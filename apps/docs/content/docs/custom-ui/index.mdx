# Custom UI Library

خوش آمدید به داکیومنت کتابخانه Custom UI! این کتابخانه مجموعه‌ای از کامپوننت‌ها، هوک‌ها و ابزارهای مفید برای توسعه اپلیکیشن‌های React است.

## ویژگی‌های کلیدی

- **کامپوننت‌های فرم**: کامپوننت‌های آماده برای React Hook Form
- **مدیریت خطا**: سیستم جامع نمایش و مدیریت خطاها
- **مدال‌ها**: کامپوننت‌های مدال responsive و قابل تنظیم
- **هوک‌های مفید**: مجموعه‌ای از هوک‌های کاربردی
- **Context ها**: مدیریت state سراسری برای خطا، مدال و toast

## شروع سریع

```tsx
import { 
  CustomButton, 
  RHFInput, 
  ErrorProvider,
  useModal 
} from '@workspace/custom-ui'

function MyComponent() {
  const { openModal } = useModal()
  
  return (
    <ErrorProvider>
      <RHFInput name="email" placeholder="ایمیل خود را وارد کنید" />
      <CustomButton onClick={() => openModal('example')}>
        باز کردن مدال
      </CustomButton>
    </ErrorProvider>
  )
}
```

## نصب

```bash
# در پروژه‌های workspace
pnpm add @workspace/custom-ui

# یا در پروژه‌های خارجی
npm install @workspace/custom-ui
```

## وابستگی‌ها

این کتابخانه به پکیج‌های زیر وابسته است:

- `@workspace/ui` - کتابخانه UI پایه
- `react-hook-form` - مدیریت فرم‌ها
- `lucide-react` - آیکون‌ها
