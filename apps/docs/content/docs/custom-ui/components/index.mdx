## کامپوننت‌های اصلی

### CustomButton

دکمه سفارشی با قابلیت‌های پیشرفته:

```tsx
import { CustomButton } from '@workspace/custom-ui'

<CustomButton 
  variant="primary" 
  size="lg"
  loading={isLoading}
  onClick={handleClick}
>
  کلیک کنید
</CustomButton>
```

### RHFInput

کامپوننت ورودی متصل به React Hook Form:

```tsx
import { RHFInput } from '@workspace/custom-ui'

<RHFInput
  name="email"
  type="email"
  placeholder="ایمیل خود را وارد کنید"
  validation={{ required: true, pattern: /^[^\s@]+@[^\s@]+\.[^\s@]+$/ }}
/>
```

### DialogModal

مدال responsive با قابلیت‌های پیشرفته:

```tsx
import { DialogModal } from '@workspace/custom-ui'

<DialogModal
  isOpen={isOpen}
  onClose={onClose}
  title="عنوان مدال"
  size="md"
>
  محتوای مدال
</DialogModal>
```
