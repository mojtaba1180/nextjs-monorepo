# کامپوننت‌های دکمه

کامپوننت‌های دکمه این کتابخانه برای ایجاد دکمه‌های زیبا و کاربردی طراحی شده‌اند.

## CustomButton

دکمه سفارشی با قابلیت‌های پیشرفته و انیمیشن‌های زیبا.

### ویژگی‌ها

- **Loading State**: نمایش وضعیت loading
- **Disabled State**: مدیریت وضعیت غیرفعال
- **Variants**: انواع مختلف ظاهر
- **Sizes**: اندازه‌های مختلف
- **Icons**: پشتیبانی از آیکون‌ها
- **Animation**: انیمیشن‌های زیبا

### استفاده پایه

```tsx
import { CustomButton } from '@workspace/custom-ui'

function BasicButtonExample() {
  return (
    <div className="space-x-2">
      <CustomButton>دکمه عادی</CustomButton>
      <CustomButton variant="primary">دکمه اصلی</CustomButton>
      <CustomButton variant="secondary">دکمه ثانویه</CustomButton>
    </div>
  )
}
```

### دکمه با وضعیت Loading

```tsx
import { useState } from 'react'

function LoadingButtonExample() {
  const [isLoading, setIsLoading] = useState(false)

  const handleClick = async () => {
    setIsLoading(true)
    
    // شبیه‌سازی عملیات async
    await new Promise(resolve => setTimeout(resolve, 2000))
    
    setIsLoading(false)
  }

  return (
    <div className="space-x-2">
      <CustomButton 
        loading={isLoading}
        onClick={handleClick}
        variant="primary"
      >
        {isLoading ? 'در حال پردازش...' : 'ارسال'}
      </CustomButton>
      
      <CustomButton 
        loading={isLoading}
        loadingText="در حال ذخیره..."
        onClick={handleClick}
        variant="success"
      >
        ذخیره
      </CustomButton>
    </div>
  )
}
```

## RHFButton

دکمه submit متصل به React Hook Form با نمایش وضعیت loading.

### ویژگی‌ها

- **Form Integration**: یکپارچگی با React Hook Form
- **Loading State**: نمایش وضعیت loading خودکار
- **Validation**: بررسی اعتبار فرم
- **Error Handling**: مدیریت خطاها

### استفاده پایه

```tsx
import { RHFButton, FormProvider } from '@workspace/custom-ui'
import { useForm } from 'react-hook-form'

function FormWithRHFButton() {
  const methods = useForm()

  const onSubmit = async (data) => {
    // شبیه‌سازی عملیات async
    await new Promise(resolve => setTimeout(resolve, 2000))
    console.log(data)
  }

  return (
    <FormProvider methods={methods} onSubmit={onSubmit}>
      <div className="space-y-4">
        <input name="name" placeholder="نام" />
        <input name="email" placeholder="ایمیل" />
        
        <RHFButton type="submit" variant="primary">
          ارسال فرم
        </RHFButton>
      </div>
    </FormProvider>
  )
}
```

## مثال‌های کاربردی

### دکمه‌های عملیات

```tsx
function ActionButtons() {
  const [isDeleting, setIsDeleting] = useState(false)
  const [isSaving, setIsSaving] = useState(false)

  const handleSave = async () => {
    setIsSaving(true)
    await new Promise(resolve => setTimeout(resolve, 1500))
    setIsSaving(false)
  }

  const handleDelete = async () => {
    setIsDeleting(true)
    await new Promise(resolve => setTimeout(resolve, 2000))
    setIsDeleting(false)
  }

  return (
    <div className="space-x-2">
      <CustomButton 
        variant="success"
        loading={isSaving}
        onClick={handleSave}
        icon={<SaveIcon className="h-4 w-4" />}
      >
        ذخیره
      </CustomButton>
      
      <CustomButton 
        variant="danger"
        loading={isDeleting}
        onClick={handleDelete}
        icon={<TrashIcon className="h-4 w-4" />}
      >
        حذف
      </CustomButton>
      
      <CustomButton 
        variant="outline"
        icon={<EditIcon className="h-4 w-4" />}
      >
        ویرایش
      </CustomButton>
    </div>
  )
}
```

### دکمه‌های ناوبری

```tsx
function NavigationButtons() {
  const [currentStep, setCurrentStep] = useState(1)
  const totalSteps = 3

  const handleNext = () => {
    if (currentStep < totalSteps) {
      setCurrentStep(currentStep + 1)
    }
  }

  const handlePrevious = () => {
    if (currentStep > 1) {
      setCurrentStep(currentStep - 1)
    }
  }

  return (
    <div className="flex justify-between items-center">
      <CustomButton 
        variant="outline"
        onClick={handlePrevious}
        disabled={currentStep === 1}
        icon={<ChevronLeftIcon className="h-4 w-4" />}
      >
        قبلی
      </CustomButton>
      
      <span className="text-sm text-gray-500">
        مرحله {currentStep} از {totalSteps}
      </span>
      
      <CustomButton 
        variant="primary"
        onClick={handleNext}
        disabled={currentStep === totalSteps}
        icon={<ChevronRightIcon className="h-4 w-4" />}
      >
        بعدی
      </CustomButton>
    </div>
  )
}
```

### دکمه‌های تایید

```tsx
function ConfirmationButtons() {
  const [showConfirm, setShowConfirm] = useState(false)

  const handleDelete = () => {
    setShowConfirm(true)
  }

  const confirmDelete = () => {
    // منطق حذف
    console.log('حذف شد')
    setShowConfirm(false)
  }

  return (
    <div>
      {!showConfirm ? (
        <CustomButton 
          variant="danger"
          onClick={handleDelete}
          icon={<TrashIcon className="h-4 w-4" />}
        >
          حذف
        </CustomButton>
      ) : (
        <div className="space-x-2">
          <CustomButton 
            variant="danger"
            onClick={confirmDelete}
            icon={<CheckIcon className="h-4 w-4" />}
          >
            تایید حذف
          </CustomButton>
          
          <CustomButton 
            variant="outline"
            onClick={() => setShowConfirm(false)}
            icon={<XIcon className="h-4 w-4" />}
          >
            انصراف
          </CustomButton>
        </div>
      )}
    </div>
  )
}
```

### دکمه‌های گروهی

```tsx
function ButtonGroup() {
  const [activeFilter, setActiveFilter] = useState('all')

  const filters = [
    { key: 'all', label: 'همه' },
    { key: 'active', label: 'فعال' },
    { key: 'inactive', label: 'غیرفعال' },
    { key: 'pending', label: 'در انتظار' }
  ]

  return (
    <div className="flex space-x-1 p-1 bg-gray-100 rounded-lg">
      {filters.map(filter => (
        <CustomButton
          key={filter.key}
          variant={activeFilter === filter.key ? 'primary' : 'ghost'}
          size="sm"
          onClick={() => setActiveFilter(filter.key)}
        >
          {filter.label}
        </CustomButton>
      ))}
    </div>
  )
}
```
